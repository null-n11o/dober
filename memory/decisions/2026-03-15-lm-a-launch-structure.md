# 決定: LM-Aシリーズローンチ構造

日付: 2026-03-15

## 決定内容

LM-Aを単体の無料LMではなく、3部構成のシリーズとして設計。①のみ無料、②③は有料商品とする。

```
① 総合値（無料LM）← メールリスト獲得
② マサラタウンから始めるレベル上げ（¥980〜¥1,980）← 初課金（tripwire）
③ ○○（低〜中単価）← 未設計
↓
本命商品（恋愛OS / 金融OS バンドル）
```

## 決定の根拠

- ①は"what"（概念提示）→ 無料で惜しみなく渡す
- ②は"how"（具体的方法論）→ 価値が高く有料にする根拠がある
- 一度課金した読者は次の商品も買いやすい（yes ladder）
- ②の価格はtripwire設計（¥980〜¥1,980）でバリアを下げる

## 制約・条件

- ②の着手タイミング: メールリスト100人超え後
- ①の公開が先決

- ROADMAP.mdのタスク一覧に反映済み
