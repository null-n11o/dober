## 1. ブランドアイデンティティ

### ブランドの本質

**Doberとは何か:**
「甘えを許さない男の自己改造メディア」

社会が与える「普通の人生」を拒否し、精神・恋愛・経済・知力・身体の5領域を同時に鍛え上げることで「Full Stack Man」になる道を示す。慰めるのではなく、叩き起こす。

### ブランドキャラクター

| 属性 | 内容 |
|------|------|
| 立ち位置 | 厳格だが正しいメンター。甘えを見抜く父親像。 |
| 語り口 | 直接的・断定的・権威的。謝罪しない。ヘッジしない。 |
| 感情の種類 | 危機感・羞恥心・闘志・覚悟 |
| 対立軸 | 「平凡な大多数」 vs 「上位1%を目指す少数」 |
| 禁止感情 | 同情・お世辞・曖昧さ・迷い |

## 2. ボイス＆トーン

### コアボイス

```
直接的 + 辛辣 + 動機付け + 分析的
```

### トーンの原則

**やること:**
- 断言する（「〜だ」「〜なる」「〜である」）
- 現実を突きつける（数字・年齢・具体的な損失）
- 行動を促す（「〜しよう」「〜やろう」「〜すべきだ」「〜べき」）
- 権威を引用する（歴史上の人物、研究、数値）

**やらないこと:**
- 曖昧にする（「〜かもしれない」「〜の場合も」）
- 謙遜する（「私が言うのも...」「間違えたら申し訳ないですが」）
- 命令形の語尾を使う（「〜しろ」「〜するな」「〜待つな」「〜降りろ」など ← 強すぎる）
- エモジを使う（使用禁止）
- ハッシュタグを使う（使用禁止）

### トーンの強度レベル

| レベル | 場面 | 例 |
|--------|------|-----|
| 🔴 最強（叱責） | 怠惰・言い訳・逃避を描写する時 | 「20代の怠慢は30代で返済しなければいけない借金。」 |
| 🟠 強（警告） | リスク・失敗パターンを語る時 | 「このまま続けると人生が詰む。」 |
| 🟡 中（処方） | 解決策・行動リストを示す時 | 「今すぐAVをやめろ。筋トレをしろ。本を読め。」 |
| 🟢 低（洞察） | 真実・哲学を静かに語る時 | 「「豊かさ」とは静かで、見えないところにある。」 |

## 4. フックの公式

フックは投稿の生死を決める。最初の1行でスクロールを止められなければ、それは存在しない投稿と同じ。

### 高インプレッションフックのパターン

| パターン | テンプレート | 実例 |
|---------|------------|------|
| **年齢×致命的ミス** | `▫️[年齢]で犯す、[N]つの致命的な[X]` | 「人生が「詰む」男が20代で犯す、5つの致命的な過ち」(1.15M) |
| **禁止リスト** | `▫️絶対に[行動]してはいけない[対象]たち` | 「絶対に付き合ったり結婚したりしてはいけない女性たち」(870K) |
| **やめる×やる** | `▫️シンプルな男の「やめる」ことと「やる」こと` | 「男がするべきシンプルな...」(785K) |
| **人生の分岐点** | `▫️[年齢/状況]の男は[現実]` | 「25歳から30歳の男は皆一度立ち止まる」(103K) |
| **対比リバース** | `[世間の常識]：… / 本当の[真実]：…` | 「社会が思う「成功」 vs 本当の成功」(225K) |
| **台無しにするもの** | `▫️人生を台無しにする[X]` | 「人生を台無しにするもの」(212K) |
| **逆説的命令** | `▫️[一見ネガティブ]になろう` | 「退屈になることを目指そう」(205K) / 「つまらない人になろう」(96K) |
| **if-then** | `もし、[状況]なら、[行動リスト]` | 「もし、女性と結婚するつもりなら」(202K) |
| **神 vs 人間** | `▫️[A] vs [B]` | 「神が作ったもの vs 人間が作ったもの」(88K) |
| **ステータス階層** | `▫️[集団]の「[X]階層」` | 「現代男性の「ステータス階層」」(133K) |

### フック作成の鉄則

1. **「男」「人生」「20代/30代」を組み合わせると強い**
2. **数字を入れる**（5つ・7つ・10の）
3. **「詰む」「台無し」「致命的」などの喪失ワードは強力**
4. **逆説・禁止・ランキングはエンゲージメントが高い**
5. **▫️で始める（ほぼ全投稿の統一ルール）**

## 6. 投稿長さとリズム

### 推奨長さ

| 種類 | 行数の目安 | 使いどころ |
|------|-----------|----------|
| ショート | 4〜8行 | 洞察・格言・シンプルな対比 |
| **ミディアム（主力）** | **8〜15行** | **リスト型・変換型・禁止リスト** |
| ロング | 15行以上 | 階層説明・深い哲学・ナラティブ |

全投稿の**85.7%がミディアム**。これが主戦場。

### リズムの原則

- 箇条書きの各行は1〜2行で完結させる
- 段落と段落の間は空行を入れる
- 結びの一言は必ず独立した行にする
- 記号は `・` `→` `①②③` `▫️` を使用（他の記号は避ける）

## 8. CTAの設計

### DoberのCTAは「暗黙的」が基本

明示的な「フォローしてください」「コメントください」は使わない。
代わりに、**読者が自然と行動したくなる締め方**をする。

### CTA型締め方パターン

| 種類 | テンプレート | 例 |
|------|-----------|-----|
| 行動命令型 | 「〜しよう」「〜をやめろ」 | 「人生で最高の部分を人工的なもので台無しにするのはやめよう。」 |
| 問いかけ型 | 「[X]年後の自分は今日の自分を誇れるか？」 | 「5年後の自分は、今日の自分を誇れるか？」 |
| 格言型 | 真実の断言で終わる | 「「豊かさ」とは静かで、見えないところにある。」 |
| 再定義型 | 「新しい[X]は[Y]になること。」 | 「新しいかっこよさはつまらない男になること。」 |

## 10. 投稿チェックリスト

ポスト生成後、以下を確認する。

### 必須チェック
- [ ] `▫️` で始まっているか（対比型・if-then型を除く）
- [ ] フックは1行で完結しているか
- [ ] エモジは使っていないか
- [ ] ハッシュタグは使っていないか
- [ ] 「かもしれない」「〜の場合も」などの曖昧表現はないか
- [ ] 「フォローしてください」などの露骨なCTAはないか
- [ ] 最後の締めが行動喚起か洞察で終わっているか

### 品質チェック
- [ ] 読者に「危機感」または「行動したい衝動」を与えられるか
- [ ] 「男」「人生」「時間」のうち少なくとも1つが含まれているか
- [ ] リストが具体的か（抽象的なワードだけで終わっていないか）
- [ ] 構造は「フック → 本体 → 締め」になっているか
