# Dober ポストリライトスキル

インプレッション3000以上のトップポストを取得し、Doberのブランドトーンを維持しながら新規ポストをリライト生成します。

## 前提知識

作業前に必ず読む:
- `businesses/dober/CONTEXT.md` — ブランド定義・ターゲット・コンテンツカテゴリ

**ブランドトーン:**
- 20〜30代男性向け、厳しく・直接的・喝を入れるスタイル
- 共感ではなく「現実直視」「行動喚起」が軸
- 「〜すべき」「〜するな」「お前は」などの直接的な語りかけ
- 理論より体験・結果ベースの表現

**5コンテンツカテゴリ:**
1. 精神・規律（喝/習慣化/ドーパミンデトックス）
2. 社交・恋愛（女性心理/アプリ戦略/LTR維持）
3. 経済・事業（資本主義攻略/投資/事業構築）
4. 知力・OS（学習/AI/自己分析）
5. 身体能力（筋トレ/食事バイオハック）

## 実行手順

### Step 1: トップポスト取得

```bash
python3 skills/dober-post-rewrite/scripts/fetch_top_posts.py --min-impressions 3000 --json
```

取得結果を確認し、ポストの傾向を分析する:
- どのカテゴリが多いか
- 共通する文体・構成パターン
- フックになっている表現・切り口

### Step 2: リライトポスト生成

取得したトップポストを元に、以下のルールで新規ポストを生成する。

**生成ルール:**
- 1つの元ポストから最大2〜3本のリライトを生成
- 合計30本を目標（ユーザーから件数指定があればそれに従う）
- 同じ内容でも「切り口・フック・締め方」を変えてバリエーションを作る
- 元の訴求軸（喝/事実提示/行動指示）は維持する
- 文字数はThreads向け: 100〜300文字が目安
- LM誘導を含めてよい（「〇〇とリプ」の形式）

**リライトパターン例:**
- 「問いかけ型」→「断言型」に変換
- 箇条書き → ストーリー型に変換
- 結論から入る → 問題提起から入る形式に変換
- カテゴリをまたいだ切り口の追加（例: 恋愛ポスト → 規律の文脈で語り直す）

**出力形式（JSON）:**
```json
[
  {
    "content": "投稿本文（100〜300文字）",
    "source_post_id": "元のThreads Post ID（わかる場合）",
    "category": "1-1" // コンテンツカテゴリコード
  },
  ...
]
```

### Step 3: 結果の保存

生成したポストをJSONファイルに保存する:

```bash
# 保存先ファイルを作成（日付付き）
OUTFILE="businesses/dober/content/drafts/$(date +%Y-%m-%d)_rewrite_posts.json"
mkdir -p businesses/dober/content/drafts/
```

生成したJSONをそのファイルに書き込む。

### Step 4: 完了報告

以下をサマリーとして報告する:
- 元にしたトップポスト数
- 生成したポスト数（カテゴリ別内訳）
- 保存先ファイルパス
- 次のステップ（`/dober-post-to-notion` でNotionに登録）

## 注意事項

- 既存ポストのコピーは厳禁。必ず新規の表現でリライトする
- 「炎上リスクのある表現」（差別・侮辱・断定的な誤情報）は使わない
- フォロワー5,000・メールリスト1,000のKPIを意識したフック設計にする
