# 決定: ブランドガイドライン v1.0 策定

日付: 2026-03-14

## 決定内容

778件のポスト実績データ（2024〜2026年）を全件分析し、`dober/assets/brand-guideline.md` を作成。

## 決定の根拠

- これまでポスト生成をAgentに依頼するたびに文体がブレていた
- 上位インプレッションポストのパターンを体系化することで再現性を高める

## 主要な知見


- **最強フックパターン**: 「年齢×致命的ミス」（1.15M imp達成）
- **最頻構造**: ▫️フック → 箇条書き → 締めの一言（全体の85%）
- **絶対禁止**: エモジ・ハッシュタグ・直接CTA・曖昧表現
- **変換型**（現状 = ネガ → 理想）が高エンゲージ

## 今後への影響
