# Dober SEOキーワードリスト化スキル

noteで書く記事テーマを受け取り、狙うべきSEOキーワードを整理する。
このスキルの出力は `dober-article-outline` のインプットになる。

## インプット

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 記事テーマ | ○ | 書きたい記事の大まかなテーマ（例:「筋トレ 始め方」「マッチングアプリ 攻略」） |
| ターゲット読者 | △ | 省略時はDoberデフォルト（20〜30代男性） |

## 実行手順

### Step 1: テーマの検索意図を分類する

テーマを以下の3つの検索意図に分類する:

| 意図 | 説明 | 例 |
|------|------|----|
| **Know（知りたい）** | 情報収集・概念理解 | 「テストステロン 上げ方 食事」 |
| **Do（やりたい）** | 具体的な方法・手順 | 「筋トレ メニュー 初心者」 |
| **Compare（比べたい）** | 選択肢の比較・判断 | 「マッチングアプリ おすすめ 男性」 |

### Step 2: Webサーチでキーワードを収集する

以下の切り口でWebサーチを実行し、実際に検索されているキーワードを収集する:

- `[テーマ] 方法`
- `[テーマ] やり方`
- `[テーマ] おすすめ`
- `[テーマ] 初心者`
- `[テーマ] コツ`
- 関連する「よくある質問（Q&A）」形式のキーワード

### Step 3: キーワードを整理・評価する

収集したキーワードを以下の基準で評価する:

**採用基準:**
- Doberのターゲット（20〜30代男性）が実際に検索しそうか
- Doberのコンテンツカテゴリ（精神・恋愛・経済・知力・身体）と合致するか
- 検索ボリュームがある程度見込める（ニッチすぎない）か

**除外基準:**
- ターゲットと明らかにズレている（女性向け・子供向けなど）
- Doberのブランドと相容れないテーマ（自己啓発の薄い内容など）

### Step 4: 出力

以下のフォーマットで出力する:

```markdown
## SEOキーワードリスト: [テーマ]

### メインキーワード（記事タイトルに入れる）
- [キーワード1]
- [キーワード2]

### サブキーワード（本文・見出しに自然に含める）
- [キーワード3]
- [キーワード4]
- [キーワード5]

### ロングテールキーワード（競合が少なく狙いやすい）
- [キーワード6]
- [キーワード7]

### 関連する検索意図（アウトライン設計のヒント）
- [読者が本当に知りたいこと1]
- [読者が本当に知りたいこと2]
- [読者が本当に知りたいこと3]

---
### dober-article-outline への引き継ぎメモ
- 推奨メインキーワード: [1つに絞る]
- 記事で必ず答えるべき問い: [読者の最大の検索意図]
- 避けるべきキーワード: [Doberブランドと相性が悪いもの]
```

## エラーハンドリング

| 状況 | 対処 |
|------|------|
| テーマが広すぎる | 「どの側面に絞るか」を確認し、テーマを1つに絞ってから進む |
| Webサーチで有効な結果が出ない | テーマをより一般的なワードに言い換えてリトライ |
| LMに対してこのスキルを呼ばれた | 「LMにSEOキーワードは不要です。dober-article-outlineに直接進んでください」と案内する |
